# Getting Started with lack

Welcome to the **LLM Attack Construction Kit (lack)** documentation.

## Installation

You can download the latest version of lack from our [website](https://lackrt.com).

### macOS

For macOS users, we support both Apple Silicon and Intel chips.

```bash
# Example command (if applicable)
brew install lack-cli
```

### Windows

Download the installer from the website and follow the instructions.

## Quick Start

Once installed, you can start a new red teaming session:

1. Open lack.
2. Select your target LLM.
3. Choose a test suite.
4. Click "Start Attack".

## Features

- **Cross-Platform**: Works on Windows and macOS.
- **Secure**: Your data is isolated.
- **High Performance**: Optimized for speed.

For more details, check out the [Features](/features) page.
