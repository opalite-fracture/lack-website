# Installation

## Prerequisites

- Node.js 18+
- Python 3.10+

## Installation Steps

1.  Clone the repository:
    ```bash
    git clone https://github.com/lackrt/lack.git
    ```
2.  Install dependencies:
    ```bash
    cd lack
    npm install
    ```
3.  Build the project:
    ```bash
    npm run build
    ```

## Verification

Run the following command to verify installation:

```bash
lack --version
```
