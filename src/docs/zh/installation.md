# 安装指南

##前提条件

- Node.js 18+
- Python 3.10+

## 安装步骤

1.  克隆仓库:
    ```bash
    git clone https://github.com/lackrt/lack.git
    ```
2.  安装依赖:
    ```bash
    cd lack
    npm install
    ```
3.  构建项目:
    ```bash
    npm run build
    ```

## 验证

运行以下命令验证安装:

```bash
lack --version
```
