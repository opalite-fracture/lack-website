# lack 快速开始

欢迎阅读 **LLM Attack Construction Kit (lack)** 文档。

## 安装

您可以从我们的 [网站](https://lackrt.com) 下载 lack 的最新版本。

### macOS

对于 macOS 用户，我们同时支持 Apple Silicon 和 Intel 芯片。

```bash
# 示例命令 (如果适用)
brew install lack-cli
```

### Windows

从网站下载安装程序并按照说明进行操作。

## 快速开始

安装完成后，您可以开始一个新的红队测试会话：

1. 打开 lack。
2. 选择您的目标 LLM。
3. 选择一个测试套件。
4. 点击 "开始攻击"。

## 功能特性

- **跨平台**: 支持 Windows 和 macOS。
- **安全**: 您的数据是隔离的。
- **高性能**: 针对速度进行了优化。

更多详情，请查看 [功能特性](/features) 页面。
